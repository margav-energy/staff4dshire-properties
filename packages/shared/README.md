# Staff4dshire Shared Package

This package contains shared code used by both the Admin Portal and Staff App.

## Contents

- **Core Models**: User, Company, Project, etc.
- **API Services**: HTTP clients, API endpoints
- **Providers**: AuthProvider, UserProvider, etc.
- **Utilities**: Helpers, constants, validators
- **Config**: API configuration, app settings

## Usage

Add this package to your app's `pubspec.yaml`:

```yaml
dependencies:
  staff4dshire_shared:
    path: ../packages/shared
```

## Development

This package is not published - it's used locally as part of the monorepo structure.
