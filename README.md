# Staff4dshire Properties

A comprehensive mobile and web application for staff and subcontractor sign-in/out, timesheet automation, and safety compliance management.

## Project Structure

```
Staff4dshire Properties/
├── mobile/              # Flutter mobile application
├── web/                 # Next.js web application
├── backend/             # PostgreSQL database schema and API documentation
├── docs/                # Design documentation and specifications
└── README.md
```

## Tech Stack

- **Mobile**: Flutter (Dart)
- **Web**: Next.js (React/TypeScript)
- **Database**: PostgreSQL
- **Design**: Material Design 3 principles with custom branding

## Features

### Core Features
- ✅ One-tap sign in/out with GPS and timestamp
- ✅ Project/site selection
- ✅ Automatic weekly timesheets with export options
- ✅ Daily fit-to-work declarations
- ✅ Digital RAMS sign-off
- ✅ Toolbox talk attendance tracking
- ✅ Document hub (CSCS, H&S certs, insurance, CPP, RAMS) with expiry reminders
- ✅ Supervisor tools: live headcount, fire roll call, approve/edit times
- ✅ Admin dashboard: attendance reports, timesheet exports, induction management

### Optional Features
- ⭐ Geofencing
- ⭐ Photo sign-in
- ⭐ AI document reading

## Getting Started

### Mobile App (Flutter)
```bash
cd mobile
flutter pub get
flutter run
```

### Web App (Next.js)
```bash
cd web
npm install
npm run dev
```
